凡例

  • : 招待制（登録にはインスタンス管理者が発行したコードが必要）/ : 新規登録可
  • : ローカルタイムラインが有効
  • : 検索が有効
  • : reCAPTCHAによるボット防御が有効
  • : ドライブのファイルはオブジェクトストレージで管理されている
  • : Twitter連携を利用可能
  • : GitHub連携を利用可能
  • : Discord連携を利用可能
  • : 通知を常時配信可能 (iOSなど、利用環境によっては利用不可)
  • : Eメール連携を利用可能
  • : インスタンスに登録されたユーザー数
  • : インスタンス内のユーザーのノートの総数
  • : 文字数制限
  • : 1ユーザーが利用できるドライブ容量
  • : インスタンス管理者名
